# Installation
The Installation Is Easy,You Just Follow This Command

      $ git clone https://github.com/KaizIqbal/Bibata_Cursor.git
      $ cd Bibata_Cursor/
      $ chmod +x build.sh
      $ ./build.sh
      $ chmod +x ./Installer_Bibata.sh
      $ sudo ./Installer_Bibata.sh
